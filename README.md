# sitemap.duyet.net
Global sitemap for [http://duyet.net](http://duyet.net)

Version 0.1 - beta 1

# How it work

1. Update sitemap link in file `sitemap.json`
2. Build by **Grunt task**

	```sh
	$ grunt 
	```

3. Publish to Github pages.
4. Sitemap: [https://sitemap.duyet.net](https://sitemap.duyet.net)
